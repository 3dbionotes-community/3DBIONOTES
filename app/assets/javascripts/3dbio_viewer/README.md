Viewer component for 3dbionotes.

## Setup

```shell
$ yarn install
$ git clone https://github.com/PDBeurope/protvista-pdb
$ (cd protvista-pdb && npm install && npm run build && yarn link)
$ yarn link provista-pdb
```

## Build

```shell
$ $ yarn build
```

## Development

```shell
$ PORT=3001 yarn start
```
